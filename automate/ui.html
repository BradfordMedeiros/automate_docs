User Interface

Description

The user interface is important in automate. The idea of automate's user interface is to act as a type of command and control software to your devices.
The user interface exposes your home automation setups status.

The UI, like the rest of automate, is still work-in-progress. While this is in no way the final look, feel, or functionality, here is a preview screenshot with actual code running.
